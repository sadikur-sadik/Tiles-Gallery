# Tiles Gallery Project

## Description

The project is about tiles where various kinds of tiles are being displayed and sold by the author. Interesting animations and precise routes make this website attractive towards the visitiors.

## Technologies

- HTML
- CSS 
- JavaScript (ES6)
- Next JS (framework)
- Tailwind CSS
- React Icons
- Better Auth
- MongoDB
- Vercel
- JSON server
- AnimateCSS

## Key Features

- This project is entirely handled by Next JS. There are three major routes (Home, All-tiles, My profile).
- Cool animations, user friendly, and fully responsive UI.
- Secured data.
- Two types of authentication method.
- While all-tiles and home is visible to everyone, user needs to sign up to see the restricted routes like - my profile, tiles details.
- Profile edit option makes the website more interactive.
- And many more a options
